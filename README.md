    1. 數字屬性(泊位)以文字方塊輸入
    2. 其它非數字屬性以勾選方塊輸入
    3. 寫入物件及屬性值
    4. javascript

#  判斷流程簡述
1. 宣告加成變數
    CHARGE_RW
    CHARGE_STAR
    1. 迴圈判斷物件屬性是否符合規則
    2. 結束前最後判斷若有 RW 則沒有 米
        除非
        1. 10萬以上貨櫃
        2. 洲際
        CHARGE_RW >= 1 則 CHARGE_STAR = 0

# 判斷是否有米
2. 每條規則表列其 (1)泊位編號(陣列) (2)噸位規則 (3)種類規則 (4)其它條件
    1. 判斷是否存在於第一欄泊位編號
    2. 判斷是否符合第二欄噸位規則
    3. 判斷是否符合第三欄種類規則
    4. 判斷是否符合第四欄其它條件
    5. 將結果存入全域變數 CHARGE_STAR

# 判斷是否有 RW
3. 以 '#' 宣告泊位編號規則 並以欄位(空白/換行) 分隔子規則
    1. 遞迴刷子規則 符合就將 CHARGE_RW +1
